# NeXgen — required behavioral/security invariants for the canonical AGENTS.md.
#
# Each non-comment, non-empty line is a SIGNATURE: a stable substring that MUST
# appear verbatim in the canonical AGENTS.md. check_required_rules.py flags any
# that are missing. This guards the non-negotiable rules from silently
# vanishing from the bootstrap -- the "vault <-> public drift" failure class,
# where a security or behavior primitive drops out of one copy unnoticed.
#
# Scope on PURPOSE to the invariants genuinely shared by every install. Rules
# that are deliberately setup-specific (e.g. a maintainer's own filesystem
# scope, whole-home vs restricted) do NOT belong here: enforcing those across
# two intentionally-different documents would be a false positive, not a guard.
#
# When you INTENTIONALLY reword one of these, update its signature here in the
# same change. That edit is the explicit acknowledgement that a non-negotiable
# rule moved -- exactly the moment this guard exists to make visible.
